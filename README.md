# Portfolio Nguyễn Thị Việt Loan

Website portfolio học phần **Nhập môn Công nghệ số và Ứng dụng AI** của Nguyễn Thị Việt Loan — MSSV 25042185.

## Chạy local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Website đã cấu hình base path cho GitHub Pages repo `portfolio-nguyen-thi-viet-loan`.
